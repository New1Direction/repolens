Getting Started
  1. Connect one AI provider below — Google AI's free key is the fastest start.
  2. Open any GitHub / GitLab / npm / PyPI page and click the RepoLens toolbar icon.
  3. Read the Verdict, then explore — press ? on any analysis for the full guide.
